Redesign the "Expenses" page for a wealth management web app called 
"Wealthly". The current page is mostly empty states and flat KPI cards with 
zero visual storytelling. Transform it into a powerful spending intelligence 
hub that integrates deeply with the Budget page — showing live budget alerts, 
category spending velocity, and actionable insights — while matching the 
existing dashboard's design system exactly.

═══════════════════════════════════════════════════════════
CORE INTEGRATION WITH BUDGET PAGE (CRITICAL)
═══════════════════════════════════════════════════════════

The Expenses page is the CONSUMPTION side of the Budget equation:

1. Every expense entry is auto-categorized and counted against the Budget 
   page's category limits
2. Budget alerts appear prominently on this page:
   - Sticky banner at top if any category is at WARNING (75-95%) or 
     DANGER (≥95%) state
   - Red flag icon next to individual transactions that pushed a category 
     over its budget
   - Category filter pills show real-time budget status with colored dots
3. "Budget vs Actual" section is the page's hero, not an afterthought
4. Sidebar "Expenses" nav item shows notification badge with count of 
   categories in warning/danger state

INTERACTIONS & MICRO-POLISH
═══════════════════════════════════════════════════════════
- Sticky alert banner only for warnings/dangers, subtle success state 
  can be collapsed by default
- Clicking a category pill filters transactions AND updates the budget 
  vs actual section to highlight only that row
- Clicking a budget bar scrolls to & filters transactions for that category
- Red flag hover tooltip: "This transaction pushed Entertainment over 
  budget by $36"
- Over-budget categories have subtle pulse animation on their bars
- Inline quick-add: press "+" key anywhere → add expense modal opens
- Bulk actions: multi-select rows → categorize, delete, export selection
- Smart autocomplete in Add Expense modal: remembers merchants, 
  auto-suggests category based on merchant name
- Receipt upload: drag & drop image → OCR extracts amount/merchant/date
- Daily spending chart annotations draw in on scroll into view
- Empty states: friendly illustrations, not just grey text
- RTL support for Arabic
