PART 3 — AI DISTRIBUTION STRATEGY (critical architectural decision)
═══════════════════════════════════════════════════════════════════════════════

The old "AI Reports" standalone page is DELETED. AI intelligence is now 
distributed in two complementary ways:

A) CONTEXTUAL AI INSIGHTS (embedded in every relevant page)
   Every major page has a "Smart Insights" section with sparkle icon (✨) 
   and soft gradient background (blue→purple subtle). These appear where 
   the data is, not in a separate location. Examples:
   
   • Home: "You're saving 23% more this month — on track for Q2 goals"
   • Budget: "Entertainment exceeded 3 months in a row. Raise limit?"
   • Expenses: "3 untagged subscriptions detected worth $67/mo"
   • Debts: "Switch to Avalanche strategy to save $840 in interest"
   • Investments: "Your portfolio is 78% tech — consider diversifying"
   • Goals: "At current pace, Car goal finishes 2 months early ✨"
   • Net Worth: "Your net worth grew 4.3% — above your 3.2% average"

B) FLOATING AI CHAT (universal assistant)
   A floating action button (bottom-right, 56px circle, blue-600 bg, 
   sparkle icon, subtle glow) available on every page.
   
   Clicking opens a right-side drawer (480px wide on desktop, full-screen 
   on mobile) containing:
   
   • Header: "Wealthly AI" + sparkle icon, close button, clear chat icon
   • Welcome state: suggested prompts as chips
     - "How much did I spend on food this month?"
     - "Am I on track with my savings goals?"
     - "What's my biggest expense category?"
     - "Should I pay off debt or invest?"
     - "Calculate my Zakat for this year"
   • Chat history with user/AI messages
   • AI responses can embed mini-charts, transaction lists, or action buttons
     (e.g., "Apply this budget change")
   • Input at bottom with mic icon for voice input
   • Context-aware: when opened from Debts page, AI knows you're asking 
     about debts
   
   This replaces the old AI Reports page entirely and is FAR more useful.

CRITICAL EXECUTION RULES
═══════════════════════════════════════════════════════════════════════════════

1. EVERY page must use the same sidebar, same cards, same typography, same 
   colors. Zero visual inconsistency between pages.

2. Data must flow between pages in real-time:
   - Adding an expense updates Budget, Reports, Net Worth instantly
   - Adding income updates Budget "available to allocate"
   - Paying a debt updates Net Worth and Debts simultaneously
   - Editing a category renames it everywhere

3. Empty states are opportunities, not dead ends. Every empty state has 
   a friendly illustration, a clear explanation, and a primary CTA.

4. Loading states use skeleton screens, not spinners.

5. Error states are helpful and actionable, never blame the user.

6. Every money value uses tabular-nums for perfect alignment.

7. Every page must have:
   - A Smart Insights section where relevant
   - Access to the floating AI chat
   - Consistent filter/search/export affordances
   - Proper empty/loading/error states

8. Admin features are completely isolated from user features — different 
   route, different theme accent, different mental model.

9. The app must feel like one product, not a collection of pages. Every 
   interaction should feel like it belongs to a family of interactions.

10. Reference aesthetic: Linear (clean, confident) × Copilot (intelligent) 
    × Apple (warm, humane) × Monzo (colorful, approachable). Avoid 
    spreadsheet-app aesthetics (Mint old), avoid overly corporate feel 
    (Quicken), avoid childish gamification (some budget apps).
