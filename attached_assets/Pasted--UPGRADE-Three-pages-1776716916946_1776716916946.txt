CROSS-PAGE INTEGRATION RULES
═══════════════════════════════════════════════════════════════════════════════

1. Every expense entered on Expenses page immediately:
   - Updates Budget category spent amount
   - Triggers budget alert if crosses threshold
   - Appears in Transactions unified hub
   - Reduces Net Worth if from bank account

2. Every investment change (buy/sell/value update):
   - Updates Net Worth total
   - Reflects in Home dashboard wealth breakdown
   - Updates Wealth Breakdown donut

3. Every asset addition/valuation change:
   - Updates Net Worth total
   - Reflects in Home dashboard
   - If linked to debt (e.g., car loan) syncs with Debts page

4. All three pages share:
   - Same currency display settings
   - Same date/time formats
   - Same AI chat context awareness (AI knows what page you're on)
