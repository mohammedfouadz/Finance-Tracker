# Personal Financial Management System - Development Prompt

## System Overview
Build a comprehensive web-based personal financial management platform that empowers users to take complete control of their finances through intelligent income allocation, detailed expense tracking, and goal-oriented financial planning.

## Core Functionality

### 1. Income Management
- Users input their monthly income (single or multiple income sources)
- System automatically calculates and distributes income across predefined financial categories
- Support for variable income (freelancers, entrepreneurs) with averaging capabilities
- Multi-currency support with real-time conversion rates

### 2. Automated Budget Allocation System
Implement a smart allocation engine based on proven financial models (50/30/20 rule, customizable):

**Default Categories:**
- **Living Expenses (50-60%)**: Essential costs for daily life
  - Housing (rent/mortgage)
  - Food & groceries
  - Transportation
  - Utilities (electricity, water, internet)
  - Healthcare & insurance
  - Personal care
  
- **Investments (10-20%)**: Growth-focused allocations
  - Stock market portfolios
  - Cryptocurrency investments
  - Real estate investments
  - Business investments
  
- **Savings (15-25%)**: Security and future planning
  - Emergency fund (target: 3-6 months expenses)
  - Gold/precious metals
  - High-yield savings accounts
  - Fixed deposits/bonds
  - Education fund
  - Retirement savings
  
- **Discretionary/Personal (10-20%)**: Quality of life
  - Entertainment & hobbies
  - Dining out
  - Travel & vacations
  - Shopping & personal purchases

### 3. Customization Engine
- **Flexible Category Management:**
  - Add, edit, or remove main categories
  - Create unlimited sub-categories within each main category
  - Set allocation as percentage OR fixed amount
  - Define priority levels for each category
  - Set spending limits and alerts

- **Sub-Category Examples:**
  - Savings → Gold Savings → Monthly gold purchase plan
  - Investments → Stocks → Individual stock tracking
  - Living Expenses → Food → Groceries vs. Dining out

### 4. Smart Dashboard & Analytics

**Main Dashboard Components:**
- **Financial Overview Card:**
  - Total monthly income
  - Total allocated amount
  - Remaining/surplus amount
  - Budget adherence percentage

- **Interactive Visualizations:**
  - Pie chart: Income distribution across main categories
  - Bar chart: Monthly comparison (current vs. previous months)
  - Line graph: Wealth growth trajectory over time
  - Donut chart: Subcategory breakdown within each main category

- **Year-at-a-Glance:**
  - 12-month income overview
  - Monthly expense trends
  - Savings accumulation graph
  - Investment performance tracking

- **Financial Health Score:**
  - Calculate overall financial health (0-100 scale)
  - Based on: savings rate, debt ratio, emergency fund status, investment diversity

### 5. Transaction Management
- **Expense Tracking:**
  - Manual entry with category assignment
  - Receipt upload and OCR scanning
  - Recurring expense automation
  - Bank account integration (optional)

- **Transaction Features:**
  - Date, amount, category, subcategory, notes
  - Attach receipts/invoices
  - Tag system for better organization
  - Search and filter capabilities

### 6. Goal Setting & Planning
- **Financial Goals Module:**
  - Short-term goals (0-1 year): vacation, electronics, etc.
  - Medium-term goals (1-5 years): car, wedding, home down payment
  - Long-term goals (5+ years): retirement, children's education

- **Goal Tracking Features:**
  - Target amount and deadline
  - Current progress visualization
  - Suggested monthly contribution
  - Milestone celebrations
  - Goal priority ranking

### 7. Reports & Insights

**Monthly Reports:**
- Income vs. expenses summary
- Category-wise spending breakdown
- Budget variance analysis
- Savings rate calculation
- Top spending categories

**Annual Reports:**
- Year-end financial summary
- Total savings accumulated
- Investment returns
- Net worth calculation
- Year-over-year growth comparison

**Smart Insights:**
- AI-powered spending pattern analysis
- Anomaly detection (unusual expenses)
- Money-saving suggestions
- Budget optimization recommendations
- Predictive cash flow analysis

### 8. Advanced Features

**Financial Planning Tools:**
- Emergency fund calculator
- Retirement planning calculator
- Debt payoff planner
- Investment return projections
- Loan/mortgage calculator

**Alerts & Notifications:**
- Budget limit warnings
- Bill payment reminders
- Savings milestone achievements
- Unusual spending alerts
- Goal deadline reminders

**Data Security:**
- End-to-end encryption
- Two-factor authentication
- Biometric login support
- Regular automated backups
- GDPR compliance

## User Interface Requirements

### Design Principles
- **Clean & Minimal:** Uncluttered interface focusing on essential information
- **Mobile-First:** Responsive design optimized for all devices
- **Intuitive Navigation:** Clear menu structure, easy access to main features
- **Visual Hierarchy:** Important information prominently displayed
- **Color Psychology:** Green for income/savings, red for expenses, blue for investments

### Key Screens
1. **Dashboard:** Overview of financial health
2. **Income:** Add/manage income sources
3. **Budget:** Allocate and adjust budget categories
4. **Transactions:** Log and review expenses
5. **Goals:** Set and track financial objectives
6. **Reports:** Detailed analytics and insights
7. **Settings:** Customize categories, notifications, preferences

### UI Components
- Interactive charts (Chart.js, D3.js, or Recharts)
- Progress bars for goals and budget usage
- Calendar view for scheduled transactions
- Draggable elements for budget allocation
- Quick-action buttons for common tasks

## Technical Architecture

### Frontend
- **Framework:** React.js or Vue.js
- **State Management:** Redux or Vuex
- **UI Library:** Material-UI, Ant Design, or Tailwind CSS
- **Charts:** Recharts, Chart.js, or D3.js
- **Forms:** Formik or React Hook Form

### Backend
- **Server:** Node.js (Express) or Python (Django/FastAPI)
- **Database:** PostgreSQL or MongoDB
- **Authentication:** JWT tokens, OAuth 2.0
- **API:** RESTful or GraphQL
- **File Storage:** AWS S3 or similar for receipt uploads

### Key APIs & Integrations
- Currency conversion API (for multi-currency support)
- Bank integration APIs (Plaid, Yodlee) - optional
- Email/SMS notifications (SendGrid, Twilio)
- OCR service for receipt scanning (Google Vision, AWS Textract)

## Data Models

### User
- ID, email, password (hashed), name
- Preferred currency, language
- Created date, last login

### Income
- ID, user_id, source_name, amount, frequency
- Start date, end date (for temporary income)

### Category
- ID, user_id, name, type (main/sub), parent_id
- Allocation percentage, allocation amount
- Color, icon, priority

### Transaction
- ID, user_id, category_id, amount, date
- Description, receipt_url, tags
- Is recurring, recurrence pattern

### Goal
- ID, user_id, name, target_amount, deadline
- Current_amount, priority, status
- Created date

### Budget
- ID, user_id, month, year
- Category allocations, total_income
- Created date, modified date

## User Personas

### Persona 1: Young Professional
- **Age:** 25-35
- **Income:** Regular monthly salary
- **Goals:** Build emergency fund, invest for retirement, save for home
- **Pain Points:** Living paycheck to paycheck, no clear savings strategy

### Persona 2: Freelancer/Entrepreneur
- **Age:** 28-45
- **Income:** Variable, multiple sources
- **Goals:** Stabilize income, plan for taxes, grow business savings
- **Pain Points:** Irregular cash flow, difficulty budgeting

### Persona 3: Family Planner
- **Age:** 35-50
- **Income:** Household income (single or dual)
- **Goals:** Children's education, family vacation, debt reduction
- **Pain Points:** Managing multiple financial priorities, household expenses
