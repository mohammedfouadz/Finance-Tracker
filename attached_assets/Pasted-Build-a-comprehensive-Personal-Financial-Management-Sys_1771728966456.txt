Build a comprehensive Personal Financial Management System called "مُدبِّر - Mudabbir" as a full-stack web application.

TECH STACK: React.js with TypeScript, Tailwind CSS with shadcn/ui, Recharts for charts, Node.js with Express.js backend, PostgreSQL with Prisma ORM, JWT authentication with bcrypt, i18next for Arabic/English bilingual support, Lucide React icons, Google Fonts (Cairo for Arabic, Inter for English).

MODULE 1 - INCOME MANAGEMENT (إدارة الدخل):
Table view with add, edit, delete functionality.
Fields per income entry:
- المصدر / Source Name (text, required) - e.g. "راتب شركة ABC", "إيجار شقة", "عمل حر"
- نوع المصدر / Source Type (dropdown, required) - Options: Salary, Freelance, Rental Income, Business, Dividends, Side Hustle, Other
- المبلغ / Amount (number, required)
- العملة / Currency (dropdown, required) - SAR, USD, EGP, EUR etc.
- التكرار / Frequency (dropdown, required) - Monthly, Weekly, Bi-weekly, Quarterly, Annually, One-time
- تاريخ الاستلام / Date Received (date, required)
- ملاحظات / Notes (textarea, optional)
- الحالة / Status (toggle, required) - Active / Inactive
Features: Monthly income summary at top, filter by source type and date range, search functionality, export to CSV, bulk delete.

---

MODULE 2 - BUDGET ALLOCATION (توزيع الدخل):
Takes total monthly income and distributes it across categories with adjustable percentages.
Default categories:
- مصاريف المعيشة / Living Expenses: 50%, blue, 🏠 icon
- الادخار / Savings: 20%, green, 🏦 icon
- الاستثمار / Investment: 15%, purple, 📈 icon
- صندوق الطوارئ / Emergency Fund: 10%, red, 🚨 icon
- أخرى / Other: 5%, gray, ➕ icon
Features:
- Visual slider or number input for each category percentage
- Total MUST always equal 100% with warning if it doesn't
- Show calculated amount next to percentage (e.g. 50% of 10,000 = 5,000 SAR)
- "Add Custom Category" button where user enters: Name, Percentage, Icon from predefined list, Color
- User can add categories like: Charity, Entertainment, Education, Travel, etc.
- Bar chart showing allocation visually
- Comparison with previous month allocation
- "Reset to Default" button
- Save as template for future months

Table view with filters, add, edit, delete.
Fields per expense:

- المبلغ / Amount (number, required)
- العملة / Currency (dropdown, required)
- التاريخ / Date (date, required)
- طريقة الدفع / Payment Method (dropdown, optional) - Cash, Credit Card, Bank Transfer, Digital Wallet
- ملاحظات / Notes (textarea, optional)
- مرفق / Attachment (file upload, optional) - Receipt photo/PDF
Features: Monthly expense summary at top, pie chart showing expense distribution by category for current month, filter by category and date range and payment method, search by description, "Over Budget" warning if expenses in any category exceed allocated budget from Module 2, daily/weekly/monthly view toggle, export to CSV.

Table and Cards hybrid view.
